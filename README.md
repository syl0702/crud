# CRUD

1. 프로젝트 폴더 생성
2. 프로젝트 폴더로 이동/ vscode 실행
    2.1 `.gitignore`, `README.md` 생성
3. django 프로젝트 생성
```bash
django-admin startproject <pjt-name> .
```

4. 가상환경 설정
```bash
python -m venv venv
```

5. 가상환경 활성화
```bash
source venv/Scripts/activate
```

6. 가상환경에 django 설치
```bash
pip install django
```
